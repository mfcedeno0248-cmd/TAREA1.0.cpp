Este proyecto simula el uso de recursos compartidos en un sistema operativo mediante procesos y mutex.
El objetivo es observar cómo los procesos solicitan recursos, cómo se controla el acceso a la sección crítica y cómo se evita el uso simultáneo mediante el mecanismo de exclusión mutua.

El código fue desarrollado en Dev-C++, utilizando hilos y control de concurrencia.

Objetivos de la práctica: 
Comprender el funcionamiento de los procesos y los recursos en un sistema operativo,
Simular una sección crítica usando mutex,
Evitar condiciones de carrera entre proceso,
Observar el comportamiento del sistema al asignar recursos limitados.

Instrucciones para ejecutar:
Abrir Dev-C++ o cualquier otro compilador compatible con C++,
Cargar el archivo TAREA6.01.cpp,
Compilar el programa con cualquier compilador estándar de C++ (ej. TDM-GCC),
Ejecutar para observar la simulación del uso de recursos y el control del mutex.
